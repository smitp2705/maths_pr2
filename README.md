# Student Math & Probability Project

## About Project

This project uses a student exam dataset to perform probability and statistics calculations using Python.

The notebook includes:

* Data loading with Pandas
* Probability calculations
* Empirical vs theoretical probability
* Binomial distribution examples
* Basic data analysis
* Graphs and visualizations

## Files

* `Math2 (1).ipynb` → Main Jupyter Notebook
* `student_dataset.csv` → Dataset used in the project

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset Columns

* study_hours
* attendance
* group_discussion
* previous_test_score
* final_exam_pass

## How to Run

1. Open the notebook in Jupyter Notebook or Google Colab.
2. Upload the dataset file.
3. Run all cells.

## Project Goal

The goal of this project is to understand probability concepts and student performance analysis using Python.
